## Run Locally

First, clone repository and install dependencies
```bash
git clone https://github.com/F4eNn/Ink-Wise.git
```
```bash
npm install
```

## Now you are able to run developer or production server:

 - ### Production
```bash
npm run start
```
 - ### Developer
```bash
npm run dev
# or
yarn dev
```
